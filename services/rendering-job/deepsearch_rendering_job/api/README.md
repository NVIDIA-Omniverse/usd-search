A comprehensive rendering and information extraction service for 3D content and USD (Universal Scene Description) files.

This service provides:
* High-performance rendering of 3D scenes and USD files from URLs
* Intelligent caching with TTL-based invalidation for optimal performance
* Concurrent processing with configurable worker limits
* Robust error handling for various rendering scenarios (load errors, empty scenes, timeouts)
* Health monitoring and service information endpoints
* Support for multiple rendering backends with automatic fallback handling
* Asynchronous processing with semaphore-based concurrency control

The service is designed for scalable 3D content processing workflows and integrates seamlessly
with USD Search API infrastructure for advanced scene analysis and information extraction.
