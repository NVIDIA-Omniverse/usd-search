Analyze the provided images of a 3D asset and generate searchable metadata.

Think like a 3D artist searching for assets: what terms would they type?

## Analysis Steps
1. Identify the object specifically (not "chair" but "office swivel chair")
2. Note the artistic style (realistic, stylized, cartoon, etc.)
3. List the dominant colors and apparent materials
4. Consider search terms users might try

## Output Format
{metadata_types}

## Field Definitions
{metadata_definitions}

## Guidelines
- Tags are the most important field - be thorough and include synonyms
- Use lowercase for all tags, colors, and materials
- Be specific: "oak dining table" not just "table"
- Only describe what you can actually see in the images
- Quality flags should be honest - this helps users filter results

Output only valid JSON, no other text.
